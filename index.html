📖 Comment installer l'application ?

1

Autoriser les sources inconnues

Allez dans Paramètres → Sécurité → activez "Installer depuis des sources inconnues"

Sur Android 8+, activez l'option pour votre navigateur
2

Télécharger l'APK

Cliquez sur le bouton "Télécharger l'APK". Le fichier sera sauvegardé dans vos "Téléchargements".

3

Installer l'application

Ouvrez le fichier APK et cliquez sur "Installer". Attendez la fin de l'installation.

4

Lancer l'application

Une fois l'installation terminée, ouvrez l'application et connectez-vous.

❓ Questions fréquentes

📱 Comment installer l'APK sur mon téléphone ? +

Téléchargez l'APK, ouvrez le fichier et suivez les instructions. Si nécessaire, activez les "sources inconnues" dans Paramètres → Sécurité.

💳 Comment fonctionne le porte-monnaie ? +

Rechargez votre porte-monnaie via Orange Money ou Airtel Money. Utilisez-le pour payer vos courses en un clic.

🚖 Comment devenir chauffeur ? +

Téléchargez l'application chauffeur, créez un compte et soumettez vos documents. Validation sous 48h.
