📖 Comment installer l'application ?

1

Autoriser les sources inconnues

Allez dans Paramètres → Sécurité → activez "Installer depuis des sources inconnues"

Sur Android 8+, activez l'option pour votre navigateur ou gestionnaire de fichiers
2

Télécharger l'APK

Cliquez sur le bouton "Télécharger l'APK" ci-dessus. Le fichier sera sauvegardé dans votre dossier "Téléchargements".

3

Installer l'application

Ouvrez le fichier APK téléchargé et cliquez sur "Installer". Attendez la fin de l'installation.

4

Lancer l'application

Une fois l'installation terminée, ouvrez l'application depuis votre menu et connectez-vous.

❓ Questions fréquentes

📱 Comment installer l'APK sur mon téléphone ? +

Téléchargez l'APK, puis ouvrez le fichier. Si vous voyez un message "Sources inconnues", allez dans Paramètres → Sécurité et activez l'option. Puis réessayez l'installation.

💳 Comment payer avec le porte-monnaie ? +

Rechargez votre porte-monnaie via Orange Money ou Airtel Money. Vous pouvez ensuite payer vos courses directement depuis l'application.

🚖 Comment devenir chauffeur ? +

Téléchargez l'application chauffeur, créez un compte et soumettez vos documents. Notre équipe validera votre dossier dans les 48h.
